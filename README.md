### This project is a Dynamic Link Library(DLL) in windows Platform
You can use the getopt/getopt_long and other getopt functions in win32 when use this DLL.
also this DLL provide some function for Doubly linked list/ Commandline parse/System performance infomation/Hash calculation(crc32 and crc64) and windows functions(time,file,power etc..). The project is constantly updating and adding new features.
